## Product Shape

- **Platform:** Installable PWA. Primary target iOS Safari 16.4+ (Web Share Target requires this) + Android Chrome. Desktop is incidental.
- **Core loop:** Welcome (first run) → Library → New Reading → Paste/PDF → Reader (RSVP with pin-at-midpoint) → Completion.
- **Reader math unchanged from v1:** `pinPosition(length) = ceil(length / 2)`; base WPM 350; punctuation multipliers 1.55× / 2.3×; long-word cap 1.5×.
- **Palette expanded:** 11 color tokens (was 9). Additions: `--page-bg #030303`, `--surface #101014`.
- **Typography expanded:** 4 default families + 6 alternatives for reader, 4 for display, 2 for UI, 2 for mono. User-picker in Settings.
- **Zero gamification, no social, local-first.**
